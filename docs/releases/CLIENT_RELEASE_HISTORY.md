# OrbitTerm 客户端版本记录

> 覆盖平台：Windows / macOS (ARM64 + x64) / Linux / Android（iOS 使用本地临时签名安装测试）
> 维护规则：每个版本新增一条，记录用户可感知的改动与兼容说明。

## v1.0.0（2026-04-09）

### 新增与优化
- 桌面端高频符号按钮统一升级为“图标 + 文字名称”，降低新用户学习成本。
- 终端工具栏关键按钮补全可读标签：新建窗口、连接日志、命令栏、最大化/恢复、关闭当前、展开/收起 SFTP。
- 资产列表操作按钮补全文字：一键部署密钥、编辑、删除。

### 多语言
- 新增并对齐多语言文案键：`terminalCommandBar`、`terminalHostInfo`（简中 / 繁中 / 英文 / 日文）。

### 发布治理
- 版本体系统一提升至 `v1.0.0`。
- 本版本要求各平台客户端产物统一写入 `releases/v1.0.0/`，并由 `releases/latest.json` 指向最新版本。
